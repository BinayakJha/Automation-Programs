## image converter

Need to install pillow
```
pip install Pillow
```

run
```
 image-change.py --help
 
```

Usage img-change.py [filename]/[filepath] options
OPTIONS
 -jpeg 
 -jpg  
 -png 
 -webp
For example: img-change.py images/test.png -jpeg


Supports interconversion between jpg/jpeg, png  and webp images
